# SBTI-SCIENCE

**Science & Bullshit Type Indicator — 多版本趣味人格测试**

## 关于本项目

本项目是为了测试 DeepSeek V4 模型的编程能力而完成的 SBTI（Silly Big Type Indicator）复刻项目，已扩展为 5 个可选版本：

| 版本 | 图标 | 说明 |
|---|---|---|
| 科研学术版 | 🔬 | 30 种科研人格 — 文献综述机器、DDL战士、实验室卷王… |
| 职场牛马版 | 💼 | 30 种职场人格 — PPT纺织工、会议存活者、带薪摸鱼大师… |
| 年轻家长版 | 👶 | 30 种育儿人格 — 早教焦虑症、睡前拉锯战选手、晒娃狂魔… |
| 当代文科生版 | 📖 | 30 种文科人格 — 批判理论成瘾者、AI写作危机感、转行预备役… |
| 当代艺术生版 | 🎨 | 30 种艺术人格 — AI艺术危机感、作品集完美主义、毕设展焦虑… |

每个版本包含 30 种人格类型 + 31 道选择题，首页可选版本进入测试。

整个项目由 DeepSeek V4 模型在对话中逐步构建完成，涵盖架构设计、题库编写、UI 渲染、动画过渡、Bug 修复、截图分享等全部环节。

## 技术栈

- 纯 HTML/CSS/JS，无框架，无构建工具
- IIFE 模块模式
- 动态脚本加载（按需加载版本题库）
- html2canvas 截图分享

## 运行方式

直接在浏览器中打开 `index.html` 即可。
