# SBTI-SCIENCE

**Science & Bullshit Type Indicator — 科研学术版**

## 关于本项目

本项目是为了测试 DeepSeek V4 模型的编程能力而完成的 SBTI（Silly Big Type Indicator）复刻项目。

原版 SBTI 是一款风靡社交网络的趣味人格测试，本项目将其改编为「科研学术版」——将人格类型映射为 30 种科研学术原型（如「文献综述机器」「DDL 战士」「显著性星号收集者」等），通过 31 道选择题计算用户的科研人格成分。

整个项目由 DeepSeek V4 模型在对话中逐步构建完成，包括：

- 整体架构设计与模块划分
- 题库设计（31 道题目，每题 A/B/C 三个选项）
- 30 种科研人格类型的定义与描述撰写
- 加权评分引擎
- UI 渲染与页面过渡动画
- 响应式 CSS 样式
- Bug 诊断与修复
- HTML2Canvas 截图分享功能

## 技术栈

- 纯 HTML/CSS/JS，无框架，无构建工具
- 所有 JS 模块以 IIFE 模式组织

## 运行方式

直接在浏览器中打开 `index.html` 即可。
